# 播客 AI 高光助手

## 问题
长音频内容难以快速定位重点和分享。

## MVP 流程
1. 转写文本/音频输入
2. 提取高光片段
3. 生成分享文案草稿
4. 输出评估结果

## API 列表
- `POST /api/transcribe`
- `POST /api/highlights`
- `POST /api/share-card`
- `POST /api/evaluate`
